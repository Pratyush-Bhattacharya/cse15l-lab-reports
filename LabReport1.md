# Lab Report 1

## Change Directory (cd) Examples:
1: Without arguments
```
[user@sahara ~]$ cd
[user@sahara ~]$
```

2: With directory argument
```
[user@sahara ~]$ cd lecture1
[user@sahara ~/lecture1]$
```

3: With file argument
```
[user@sahara ~/lecture1]$ cd Hello.java
bash: cd: Hello.java: Not a directory
```
